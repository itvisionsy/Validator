Class Validator

The bulk validation class.

It handles the instantiation and feeding of validation items and rules, and evaluating them.

Basically, you will not need to handle other validator classes manually unless you need to customize the validation. When created, you provide it with the rules and the values, and you will get an associative array of errors if any.
